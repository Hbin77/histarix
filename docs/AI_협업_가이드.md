# Histarix AI 협업 가이드

## 1. 프로젝트 개요

Histarix는 AI와의 전면적인 협업을 통해 기획, 설계, 구현, 테스트, 배포까지 전 과정을 수행한 프로젝트입니다.

## 2. AI 활용 단계별 내역

### 2.1 기획 단계

**사용 도구**: Claude Code (Claude Opus)

- **프로젝트 컨셉 수립**: "세계 지도 기반 인터랙티브 역사 학습 플랫폼" 컨셉을 AI와 브레인스토밍
- **기술 스택 선정**: Next.js 15 + FastAPI 조합 결정, 무료 API 중심 아키텍처 설계
- **API 아키텍처**: Wikidata SPARQL, Wikipedia REST, Wikimedia On This Day 등 무료 데이터 소스 발굴 및 통합 전략 수립
- **수익 모델**: Freemium + B2B 교육기관 라이선스 모델 기획

### 2.2 설계 단계

**사용 도구**: Claude Code

- **프로젝트 구조**: Frontend/Backend 분리 구조, 컴포넌트 트리 설계
- **데이터 흐름**: 사용자 인터랙션 → API 요청 → 캐싱 → 응답 파이프라인 설계
- **스키마 설계**: Pydantic v2 기반 API 스키마, TypeScript 인터페이스
- **캐싱 전략**: Redis 기반 TTL 캐시 (국가 24h, 이벤트 1h, 오늘의역사 6h)

### 2.3 구현 단계

**사용 도구**: Claude Code (전체 코딩)

#### Frontend
- Mapbox GL JS 기반 인터랙티브 세계 지도 (다크 테마)
- 국가 상세 패널 (정보/역사/타임라인 3탭)
- 시간 슬라이더 (BC 3000 ~ AD 2026)
- 오늘의 역사 위젯
- AI 역사 가이드 챗봇 (스트리밍 UI)
- 다국어 지원 (한/영/중/일)
- 반응형 모바일 디자인

#### Backend
- FastAPI 비동기 REST API
- Wikidata SPARQL 쿼리 엔진
- Wikipedia/Wikimedia 데이터 통합
- Redis 캐싱 레이어
- OpenAI GPT API 스트리밍 챗봇
- JWT 인증 시스템

### 2.4 보안 단계

**사용 도구**: Claude Code (보안 리뷰 에이전트)

- SSRF 방지: URL 화이트리스트, 내부 IP 차단
- 캐시 포이즈닝 방지: 캐시 키 정규화
- 입력 검증: ISO 코드, QID, 날짜 범위 등 정규식 검증
- SPARQL Injection 방지
- CORS 정책 설정

### 2.5 배포 단계

- Docker Compose 기반 컨테이너화
- Nginx 리버스 프록시 설정

## 3. AI 핵심 기능: Histy (AI 역사 가이드)

### 구현 방식
- **모델**: GPT-4o-mini (OpenAI API)
- **인터페이스**: 실시간 SSE 스트리밍
- **컨텍스트 인식**: 현재 선택한 국가 정보를 시스템 프롬프트에 주입
- **다국어**: 사용자 언어 설정에 따라 해당 언어로 응답
- **히스토리**: 최대 10턴 대화 기록 유지

### 시스템 프롬프트 전략
```
역할: Histy - AI 역사 가이드
특성: 교육적, 정확, 친근한 톤
제약: 2-4 문단 이내, 불확실한 정보는 명시
국가 컨텍스트: 선택된 국가와 연관지어 답변
```

### 교육적 가치
- 학생들이 지도를 탐색하며 궁금한 역사를 즉시 질문 가능
- 국가별 맞춤 컨텍스트로 깊이 있는 학습
- 연관 주제 제안으로 자기주도 학습 촉진

## 4. AI 활용 효율성

| 지표 | 내용 |
|------|------|
| 개발 기간 | 약 1주 (기획~배포) |
| 코드 라인 수 | ~5,000+ lines |
| AI 자동화율 | 약 95% (코딩, 디버깅, 리팩토링) |
| 지원 언어 | 4개 (한/영/중/일) |
| 통합 API | 10+ 외부 API |
| 보안 조치 | 6개 주요 영역 |

## 5. 교육 현장 페인포인트 해결

| 기존 문제 | Histarix 솔루션 |
|----------|----------------|
| 역사 수업이 텍스트 중심으로 지루함 | 인터랙티브 지도 + 시각적 타임라인 |
| 시공간적 맥락 파악 어려움 | 시간 슬라이더로 시대별 변화 체험 |
| 궁금한 점을 즉시 해결 불가 | AI 챗봇으로 실시간 Q&A |
| 다국적 학습 환경 미지원 | 4개 언어 다국어 지원 |
| 역사 사건 간 연결성 부족 | AI가 관련 사건/인물 추천 |
